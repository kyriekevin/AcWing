# Command

1. ctrl c: stop the running program.
2. ctrl u: delete the whole line.
3. tab: auto complete the command.
4. ls: list the files and folders in the current directory.
5. pwd: print the current working directory.
6. cd: change the current working directory.
7. cp: copy a file or folder.
8. mkdir: create a new folder.
9. rm: remove a file or folder.
10. touch: create a new file.
11. cat: print the content of a file.
